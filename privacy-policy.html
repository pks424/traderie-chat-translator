Privacy Policy

Last updated: July 14, 2026

Traderie Chat Translator ("the Extension") is a Chrome browser extension that automatically translates chat messages on traderie.com. This Privacy Policy explains what data the Extension collects, how it is used, and how it is protected.

1. Data We Collect

The Extension collects and processes the following data solely for the purpose of translation:

  • Message text – Text content of traderie.com chat messages is sent to the translation service you select (see Section 3) to perform translation. Message text is never stored by the Extension.
  • User preferences – Language settings (source and target language), toggle preferences (auto-translate on/off, show outgoing translation on/off), glossary entries, and optionally an API key for your chosen translation provider are stored locally in your browser using chrome.storage.local. This data never leaves your device except when an API key you provided is used to call the corresponding translation service.

2. Data We Do NOT Collect

  • We do not collect, store, or transmit any personal information to our own servers. The Extension has no server of its own.
  • We do not track browsing history or activity beyond what is necessary to translate visible traderie.com chat messages.
  • We do not share any data with third parties other than the translation service you explicitly select (see Section 3).

3. Third-Party Services

Depending on the translation engine you choose in the settings, message text is sent to one of the following services:

  • Google Translate (free endpoint) – Default engine, no API key required. Subject to Google's Privacy Policy.
  • Google Cloud Translation API (optional, requires your API key) – Subject to Google Cloud Privacy Notice.
  • Google Gemini API (optional, requires your API key) – Subject to Google's Privacy Policy.
  • Anthropic Claude API (optional, requires your API key) – Subject to Anthropic's Privacy Policy.
  • OpenAI API (optional, requires your API key) – Subject to OpenAI's Privacy Policy.

Message text sent to these services is governed by their respective privacy policies and terms of service. We encourage you to review those policies.

4. Permissions Explained

  • storage – Saves your language preferences, glossary, and API keys locally in the browser.
  • host_permissions (traderie.com) – Required to read and modify traderie chat message content for translation.
  • host_permissions (translate.googleapis.com, translation.googleapis.com, generativelanguage.googleapis.com, api.anthropic.com, api.openai.com) – Required to send text to the translation service you selected and receive translated results.

5. Data Retention

Message text is processed in-memory only and is never stored. A small in-memory translation cache (cleared when the tab closes) avoids re-translating identical text. User preference settings are stored locally in your browser and can be cleared at any time by removing the Extension or clearing browser storage.

6. Children's Privacy

The Extension is not directed at children under the age of 13. We do not knowingly collect personal information from children.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date.

8. Contact

If you have questions or concerns about this Privacy Policy, please open an issue on the project's GitHub repository.
